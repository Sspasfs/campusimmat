Description détaillée

Class that handles profiling sections of code. The data is viewable in the debug bar only. If the current user is not in debug mode, the profiler is disabled.

start ( string $name,
string $category = self::CATEGORY_CORE )

Starts a new section in the profiler. This section will be stopped when Profiler::stop() is called with the same name.

Paramètres
string$nameThe name of the section. This name will be used to stop the section later.
string$categoryThe category of the section. See Profiler::CATEGORY_* for some predefined categories.
