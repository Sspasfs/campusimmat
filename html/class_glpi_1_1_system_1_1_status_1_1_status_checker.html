◆ STATUS_NO_DATA

const STATUS_NO_DATA = 'NO_DATA'

Unable to get the status of a plugin or service. This is likely due to a prerequisite plugin or service being unavailable or the plugin not implementing the status hook. For example, some checks require the DB to be accessible.
