static getSpecificities ( $specif = '')

Get the specificities of the given device. For instance, the serial number, the size of the memory, the frequency of the CPUs ...

Paramètres
$specifstring specificity to display

Should be overloaded by Item_Device*

Renvoie
array of the specificities: index is the field name and the values are the attributs of the specificity (long name, short name, size)

Réimplémentée à partir de Item_Devices.
