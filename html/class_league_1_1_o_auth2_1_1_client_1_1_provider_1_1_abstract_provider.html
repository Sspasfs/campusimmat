Description détaillée

Represents a service provider (authorization server).

Roles (RFC 6749, §1.1)

Documentation des constructeurs et destructeur

◆ __construct()

__construct ( array $options = [],
array $collaborators = [] )

Constructs an OAuth 2.0 service provider.

Paramètres
array$optionsAn array of options to set on this provider. Options include clientId, clientSecret, redirectUri, and state. Individual providers may introduce more options, as needed.
array$collaboratorsAn array of collaborators that may be used to override this provider's default behavior. Collaborators include grantFactory, requestFactory, and httpClient. Individual providers may introduce more collaborators, as needed.
