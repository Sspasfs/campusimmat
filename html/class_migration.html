changeSearchOption ( string $itemtype,
int $old_search_opt,
int $new_search_opt )

Migrate search option values in various locations in the database. This does not change the actual search option ID. This must still be changed manually in the itemtype's class file. The changes made by this function will only be applied when the migration is finalized through Migration::executeMigration().
