bool$useSSLWhether to connect via SSL. Required when pushing messages to users that are not the pushover.net app owner. OpenSSL is required for this option.

int$retryThe retry parameter specifies how often (in seconds) the Pushover servers will send the same notification to the user.
int$expireThe expire parameter specifies how many seconds your notification will continue to be retried for (every retry seconds).
