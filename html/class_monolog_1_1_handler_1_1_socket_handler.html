Description détaillée

Stores to any socket - uses fsockopen() or pfsockopen().

Auteur
Pablo de Leon Belloc pablo.nosp@m.lb@g.nosp@m.mail..nosp@m.com
Voir également
http://php.net/manual/en/function.fsockopen.php
