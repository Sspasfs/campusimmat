Description détaillée

Collection of static methods to parse RFC strings.

This is used internally by RRule and RSet. The methods are public, BUT this isn't part of the public API of this library. Therefore there is no guarantee they will not break even for a minor version release.

Parse both DTSTART and RRULE (and EXRULE).

It's impossible to accuratly parse a RRULE in isolation (without the DTSTART) as some tests depends on DTSTART (notably the date format for UNTIL).
