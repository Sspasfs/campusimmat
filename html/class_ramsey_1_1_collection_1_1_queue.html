offsetSet ( $offset,
$value )

Sets the given value to the given offset in the array.ArrayAccess::offsetSet() array-key|null $offset The offset to set. If null, the value may be set at a numerically-indexed offset. T $value The value to set at the given offset.

Since arbitrary offsets may not be manipulated in a queue, this method serves only to fulfill the ArrayAccess interface requirements. It is invoked by other operations when adding values to the queue.
