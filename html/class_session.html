◆ buildSessionName()

static buildSessionName ( ?string $path = null,
?string $host = null,
?string $port = null )
static

Build the session name based on GLPI's folder path + full domain + port

Adding the full domain name prevent two GLPI instances on the same domain (e.g. test.domain and prod.domain) with identical folder's path (e.g. /var/www/glpi) to compete for the same cookie name

Adding the port prevent some conflicts when using docker

Paramètres
string | null$pathDefault to GLPI_ROOT
string | null$hostDefault to $_SERVER['HTTP_HOST']
string | null$portDefault to $_SERVER['SERVER_PORT']
Renvoie
string An unique session name
