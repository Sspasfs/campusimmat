◆ calculateTime()

calculateTime ( string $seconds,
string $microseconds )

Uses the provided seconds and micro-seconds to calculate the count of 100-nanosecond intervals since UTC 00:00:00.00, 15 October 1582, for RFC 4122 variant UUIDs

RFC 4122, § 4.2.2: Generation Details string $seconds A string representation of the number of seconds since the Unix epoch for the time to calculate string $microseconds A string representation of the micro-seconds associated with the time to calculate Hexadecimal The full UUID timestamp as a Hexadecimal value -pure

Implémenté dans BigNumberTimeConverter, et PhpTimeConverter.
