 HTMLTable class Create a smart HTML table. The table allows cells to depend on other ones. As such, it is possible to have rowspan for cells that are "{}father"{} of other ones. If a "{}father"{} has several sons, then, it "{}rowspans"{} on all. The table integrates the notion of group of rows (\doxylink{class_h_t_m_l_table_group}{HTMLTable\+Group}). For instance, for Computer\+\_\+\+Device, each group represents a kind of device (network card, graphique card, processor, memory, ...). There is \doxylink{class_h_t_m_l_table_super_header}{HTMLTable\+Super\+Header} that defines global headers for all groups. Each group can cut these \doxylink{class_h_t_m_l_table_super_header}{HTMLTable\+Super\+Header} as many \doxylink{class_h_t_m_l_table_sub_header}{HTMLTable\+Sub\+Header} as necessary. There is an automatic organisation of the headers between groups.

The (strict) order of definition of the table is\+:
\begin{DoxyItemize}
\item Define all \doxylink{class_h_t_m_l_table_super_header}{HTMLTable\+Super\+Header} that are used by each group\+: \doxylink{class_h_t_m_l_table_base_a83cb558a290243a38c3bae2ffc7d0822}{HTMLTable\+Main\+::add\+Header()}
\item Define one \doxylink{class_h_t_m_l_table_group}{HTMLTable\+Group}\+: \doxylink{class_h_t_m_l_table_main_af5e2aa53f1f30804e9ee736f31cf350f}{HTMLTable\+Main\+::create\+Group()}
\begin{DoxyItemize}
\item Define all \doxylink{class_h_t_m_l_table_sub_header}{HTMLTable\+Sub\+Header} depending of previously defined \doxylink{class_h_t_m_l_table_super_header}{HTMLTable\+Super\+Header} for the given group\+: \doxylink{class_h_t_m_l_table_base_a83cb558a290243a38c3bae2ffc7d0822}{HTMLTable\+Group\+::add\+Header()}
\item Create all \doxylink{class_h_t_m_l_table_row}{HTMLTable\+Row} for the given group\+: \doxylink{class_h_t_m_l_table_group_a946344c2ff4126aa70bd0487004f5d06}{HTMLTable\+Group\+::create\+Row()}
\begin{DoxyItemize}
\item Create all \doxylink{class_h_t_m_l_table_cell}{HTMLTable\+Cell} for the given row \+: HTMLTable\+Row\+::add\+Cell() and so on for each group. When done, call \doxylink{class_h_t_m_l_table_main_a68a7528fe468efd0ced934e26835b03d}{HTMLTable\+Main\+::display()} to render the table.
\end{DoxyItemize}
\end{DoxyItemize}
\end{DoxyItemize}
