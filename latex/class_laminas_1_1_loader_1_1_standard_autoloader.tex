\doxysubsection{Description détaillée}
PSR-\/0 compliant autoloader

Allows autoloading both namespaced and vendor-\/prefixed classes. Class lookups are performed on the filesystem. If a class file for the referenced class is not found, a PHP warning will be raised by \doxylink{report_8contract_8php_a77b973d137fb33212e018b042df6e3e7}{include()}.
