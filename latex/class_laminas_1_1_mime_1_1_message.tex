Generate MIME-\/compliant message from the current configuration

This can be a multipart message if more than one MIME part was added. If only one part is present, the content of this part is returned. If no part had been added, an empty string is returned.

Parts are separated by the mime boundary as defined in Laminas\textbackslash{}\+Mime\textbackslash{}\+Mime. If {\ttfamily \doxylink{}{set\+Mime()}} has been called before this method, the Laminas\textbackslash{}\+Mime\textbackslash{}\+Mime object set by this call will be used. Otherwise, a new Laminas\textbackslash{}\+Mime\textbackslash{}\+Mime object is generated and used.
