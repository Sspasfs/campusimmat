\doxysubsection{Description détaillée}
Reflection-\/based factory.

To ease development, this factory may be used for classes with type-\/hinted arguments that resolve to services in the application container; this allows omitting the step of writing a factory for each controller.

You may use it as either an abstract factory\+:

{\ttfamily  \textquotesingle{}service\+\_\+manager\textquotesingle{} =\texorpdfstring{$>$}{>} \mbox{[} \textquotesingle{}abstract\+\_\+factories\textquotesingle{} =\texorpdfstring{$>$}{>} \mbox{[} Reflection\+Based\+Abstract\+Factory\+::class, \mbox{]}, \mbox{]}, }

Or as a factory, mapping a class name to it\+:

{\ttfamily  \textquotesingle{}service\+\_\+manager\textquotesingle{} =\texorpdfstring{$>$}{>} \mbox{[} \textquotesingle{}factories\textquotesingle{} =\texorpdfstring{$>$}{>} \mbox{[} My\+Class\+With\+Dependencies\+::class =\texorpdfstring{$>$}{>} Reflection\+Based\+Abstract\+Factory\+::class, \mbox{]}, \mbox{]}, }

The latter approach is more explicit, and also more performant.

The factory has the following constraints/features\+:


\begin{DoxyItemize}
\item A parameter named {\ttfamily \$config} typehinted as an array will receive the application "{}config"{} service (i.\+e., the merged configuration).
\item Parameters type-\/hinted against array, but not named {\ttfamily \$config} will be injected with an empty array.
\item Scalar parameters will result in an exception being thrown, unless a default value is present; if the default is present, that will be used.
\item If a service cannot be found for a given typehint, the factory will raise an exception detailing this.
\item Some services provided by Laminas components do not have entries based on their class name (for historical reasons); the factory allows defining a map of these class/interface names to the corresponding service name to allow them to resolve.
\end{DoxyItemize}

{\ttfamily \$options} passed to the factory are ignored in all cases, as we cannot make assumptions about which \doxylink{report_8contract_8php_a77b973d137fb33212e018b042df6e3e7}{argument(s)} they might replace.

Based on the Lazy\+Controller\+Abstract\+Factory from laminas-\/mvc.
