\doxysubsection{Description détaillée}
Abstract plugin manager.

Abstract Plugin\+Manager\+Interface implementation providing\+:


\begin{DoxyItemize}
\item creation context support. The constructor accepts the parent container instance, which is then used when creating instances.
\item plugin validation. Implementations may define the {\ttfamily \$instance\+Of} property to indicate what class types constitute valid plugins, omitting the requirement to define the {\ttfamily validate()} method.
\end{DoxyItemize}

The implementation extends {\ttfamily Service\+Manager}, thus providing the same set of capabilities as found in that implementation.
