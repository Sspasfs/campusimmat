\doxysubsection{Description détaillée}
Factory for instantiating classes with no dependencies or which accept a single array.

The Invokable\+Factory can be used for any class that\+:


\begin{DoxyItemize}
\item has no constructor arguments;
\item accepts a single array of arguments via the constructor.
\end{DoxyItemize}

It replaces the "{}invokables"{} and "{}invokable class"{} functionality of the v2 service manager.
