\doxysubsection{Description détaillée}
Service Manager.

Default implementation of the Service\+Locator\+Interface, providing capabilities for object creation via\+:


\begin{DoxyItemize}
\item factories
\item abstract factories
\item delegator factories
\item lazy service factories (generated proxies)
\item initializers (interface injection)
\end{DoxyItemize}

It also provides the ability to inject specific service instances and to define aliases.
