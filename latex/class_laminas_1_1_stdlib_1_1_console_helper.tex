\doxysubsection{Description détaillée}
Utilities for console tooling.

Provides the following facilities\+:


\begin{DoxyItemize}
\item Colorize strings using markup (e.\+g., {\ttfamily \texorpdfstring{$<$}{<}info\texorpdfstring{$>$}{>}message\texorpdfstring{$<$}{<}/info\texorpdfstring{$>$}{>}}, {\ttfamily \texorpdfstring{$<$}{<}error\texorpdfstring{$>$}{>}message\texorpdfstring{$<$}{<}/error\texorpdfstring{$>$}{>}})
\item Write output to a specified stream, optionally with colorization.
\item Write a line of output to a specified stream, optionally with colorization, using the system EOL sequence..
\item Write an error message to STDERR.
\end{DoxyItemize}

Colorization will only occur when expected sequences are discovered, and then, only if the console terminal allows it.

Essentially, provides the bare minimum to allow you to provide messages to the current console.
