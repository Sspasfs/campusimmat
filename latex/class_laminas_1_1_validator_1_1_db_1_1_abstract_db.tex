Provides basic configuration for use with Laminas\textbackslash{}\+Validator\textbackslash{}\+Db Validators Setting \$exclude allows a single record to be excluded from matching. Exclude can either be a String containing a where clause, or an array with {\ttfamily field} and {\ttfamily value} keys to define the where clause added to the sql. A database adapter may optionally be supplied to avoid using the registered default adapter.

The following option keys are supported\+: \textquotesingle{}table\textquotesingle{} =\texorpdfstring{$>$}{>} The database table to validate against \textquotesingle{}schema\textquotesingle{} =\texorpdfstring{$>$}{>} The schema keys \textquotesingle{}field\textquotesingle{} =\texorpdfstring{$>$}{>} The field to check for a match \textquotesingle{}exclude\textquotesingle{} =\texorpdfstring{$>$}{>} An optional where clause or field/value pair to exclude from the query \textquotesingle{}adapter\textquotesingle{} =\texorpdfstring{$>$}{>} An optional database adapter to use
