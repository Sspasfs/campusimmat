\doxysubsection{Description détaillée}
Validate that a value is countable and the count meets expectations.

The validator has five specific behaviors\+:


\begin{DoxyItemize}
\item You can determine if a value is countable only
\item You can test if the value is an exact count
\item You can test if the value is greater than a minimum count value
\item You can test if the value is greater than a maximum count value
\item You can test if the value is between the minimum and maximum count values
\end{DoxyItemize}

When creating the instance or calling {\ttfamily set\+Options()}, if you specify a "{}count"{} option, specifying either "{}min"{} or "{}max"{} leads to an inconsistent state and, as such will raise an Exception\textbackslash{}\+Invalid\+Argument\+Exception.
