\doxysubsection{Description détaillée}
A reusable attribute to help configure a class or a method as a processor.

Using it offers no guarantee\+: it needs to be leveraged by a \doxylink{namespace_monolog}{Monolog} third-\/party consumer.

Using it with the \doxylink{namespace_monolog}{Monolog} library only has no effect at all\+: processors should still be turned into a callable if needed and manually pushed to the loggers and to the processable handlers.
