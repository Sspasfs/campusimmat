\doxysubsection{Description détaillée}
Channel and Error level based monolog activation strategy. Allows to trigger activation based on level per channel. e.\+g. trigger activation on level \textquotesingle{}ERROR\textquotesingle{} by default, except for records of the \textquotesingle{}sql\textquotesingle{} channel; those should trigger activation on level \textquotesingle{}WARN\textquotesingle{}.

Example\+:

{\ttfamily  \$activation\+Strategy = new Channel\+Level\+Activation\+Strategy( Logger\+::\+CRITICAL, array( \textquotesingle{}request\textquotesingle{} =\texorpdfstring{$>$}{>} Logger\+::\+ALERT, \textquotesingle{}sensitive\textquotesingle{} =\texorpdfstring{$>$}{>} Logger\+::\+ERROR, ) ); \$handler = new Fingers\+Crossed\+Handler(new Stream\+Handler(\textquotesingle{}php\+://stderr\textquotesingle{}), \$activation\+Strategy); }
