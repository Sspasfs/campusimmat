\doxysubsection{Description détaillée}
IFTTTHandler uses c\+URL to trigger IFTTT Maker actions

Register a secret key and trigger/event name at \href{https://ifttt.com/maker}{\texttt{ https\+://ifttt.\+com/maker}}

value1 will be the channel from monolog\textquotesingle{}s Logger constructor, value2 will be the level name (ERROR, WARNING, ..) value3 will be the log record\textquotesingle{}s message
