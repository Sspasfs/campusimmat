\doxysubsection{Description détaillée}
Logs to a Mongo\+DB database.

Usage example\+:

\$log = new \textbackslash{}\+Monolog\textbackslash{}\+Logger(\textquotesingle{}application\textquotesingle{}); \$client = new \textbackslash{}\+Mongo\+DB\textbackslash{}\+Client(\textquotesingle{}mongodb\+://localhost\+:27017\textquotesingle{}); \$mongodb = new \textbackslash{}\+Monolog\textbackslash{}\+Handler\textbackslash{}\+Mongo\+DBHandler(\$client, \textquotesingle{}logs\textquotesingle{}, \textquotesingle{}prod\textquotesingle{}); \$log-\/\texorpdfstring{$>$}{>}push\+Handler(\$mongodb);

The above examples uses the Mongo\+DB PHP library\textquotesingle{}s client class; however, the Mongo\+DB\textbackslash{}\+Driver\textbackslash{}\+Manager class from ext-\/mongodb is also supported.
