\doxysubsection{Description détaillée}
Proxies log messages to an existing PSR-\/3 compliant logger.

If a formatter is configured, the formatter\textquotesingle{}s output MUST be a string and the formatted message will be fed to the wrapped PSR logger instead of the original log record\textquotesingle{}s message.

\doxysubsubsection{\texorpdfstring{handle()}{handle()}}
{\footnotesize\ttfamily handle (\begin{DoxyParamCaption}\item[{array}]{\$record }\end{DoxyParamCaption})}

Handles a record.

All records may be passed to this method, and the handler should discard those that it does not want to handle.

The return value of this function controls the bubbling process of the handler stack. Unless the bubbling is interrupted (by returning true), the \doxylink{class_monolog_1_1_logger}{Logger} class will keep on calling further handlers in the stack with a given log record.


\begin{DoxyParams}[1]{Paramètres}
array & {\em \$record} & The record to handle \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Renvoie}
bool true means that this handler handled the record, and that bubbling is not permitted. false means the record was either not processed or that this handler allows bubbling.
\end{DoxyReturn}
