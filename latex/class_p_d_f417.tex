{\footnotesize\ttfamily \$\mbox{\hyperlink{report_8contract_8php_a77b973d137fb33212e018b042df6e3e7}{clusters}}\hspace{0.3cm}{\ttfamily [protected]}}

Clusters of codewords (0, 3, 6)~\newline
 Values are hex equivalents of binary representation of bars (1 = bar, 0 = space).~\newline
 The codewords numbered from 900 to 928 have special meaning, some enable to switch between modes in order to optimise the code\+:
\begin{DoxyItemize}
\item 900 \+: Switch to "{}\+Text"{} mode 
\item 901 \+: Switch to "{}\+Byte"{} mode 
\item 902 \+: Switch to "{}\+Numeric"{} mode 
\item 903 -\/ 912 \+: Reserved 
\item 913 \+: Switch to "{}\+Octet"{} only for the next codeword 
\item 914 -\/ 920 \+: Reserved 
\item 921 \+: Initialization 
\item 922 \+: Terminator codeword for Macro PDF control block 
\item 923 \+: Sequence tag to identify the beginning of optional fields in the Macro PDF control block 
\item 924 \+: Switch to "{}\+Byte"{} mode (If the total number of byte is multiple of 6) 
\item 925 \+: Identifier for a user defined Extended Channel Interpretation (ECI) 
\item 926 \+: Identifier for a general purpose ECI format 
\item 927 \+: Identifier for an ECI of a character set or code page 
\item 928 \+: Macro marker codeword to indicate the beginning of a Macro PDF Control Block 
\end{DoxyItemize}\Hypertarget{class_p_d_f417_a725ac4adfd797d8dca33964e609affcd}\label{class_p_d_f417_a725ac4adfd797d8dca33964e609affcd}
