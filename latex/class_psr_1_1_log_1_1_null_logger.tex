\doxysubsection{Description détaillée}
This Logger can be used to avoid conditional log calls.

Logging should always be optional, and if no logger is provided to your library creating a Null\+Logger instance to have something to throw logs at is a good way to avoid littering your code with {\ttfamily if (\$this-\/\texorpdfstring{$>$}{>}logger) \{ \}} blocks.
