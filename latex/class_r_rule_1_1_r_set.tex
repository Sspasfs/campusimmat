\doxysubsubsection{\texorpdfstring{getIterator()}{getIterator()}}
{\footnotesize\ttfamily get\+Iterator (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}

This method will iterate over a bunch of different iterators (rrules and arrays), keeping the results {\itshape in order}, while never attempting to merge or sort anything in memory. It can combine both finite and infinite rrule.

What we need to do it to build two heaps\+: rlist and exlist Each heap contains multiple iterators (either \doxylink{namespace_r_rule}{RRule} or Array\+Iterator) At each step of the loop, it calls all of the iterators to generate a new item, and stores them in the heap, that keeps them in order.

This is made slightly more complicated because this method is a generator.
