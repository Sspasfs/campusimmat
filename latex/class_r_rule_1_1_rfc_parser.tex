\doxysubsection{Description détaillée}
Collection of static methods to parse RFC strings.

This is used internally by \doxylink{namespace_r_rule}{RRule} and RSet. The methods are public, BUT this isn\textquotesingle{}t part of the public API of this library. Therefore there is no guarantee they will not break even for a minor version release.
