\doxysubsection{Description détaillée}
A {\ttfamily Typed\+Map} represents a map of elements where key and value are typed.

Each element is identified by a key with defined type and a value of defined type. The keys of the map must be unique. The values on the map can be= repeated but each with its own different key.

The most common case is to use a string type key, but it\textquotesingle{}s not limited to this type of keys.

This is a direct implementation of {\ttfamily Typed\+Map\+Interface}, provided for the sake of convenience.
