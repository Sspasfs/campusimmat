\doxysubsection{Description détaillée}
Thrown to indicate that the source of random data encountered an error

This exception is used mostly to indicate that \doxylink{report_8contract_8php_a77b973d137fb33212e018b042df6e3e7}{random\+\_\+bytes()} or \doxylink{report_8contract_8php_a77b973d137fb33212e018b042df6e3e7}{random\+\_\+int()} threw an exception. However, it may be used for other sources of random data.
