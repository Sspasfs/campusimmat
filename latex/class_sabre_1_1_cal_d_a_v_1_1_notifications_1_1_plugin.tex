\doxysubsection{Description détaillée}
Notifications plugin.

This plugin implements several features required by the caldav-\/notification draft specification.

Before version 2.\+1.\+0 this functionality was part of Sabre\textbackslash{}\+Cal\+DAV\textbackslash{}\+Plugin but this has since been split up.
