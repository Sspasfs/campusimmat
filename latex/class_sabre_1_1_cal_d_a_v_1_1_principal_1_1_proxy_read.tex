\doxysubsection{Description détaillée}
Proxy\+Read principal.

This class represents a principal group, hosted under the main principal. This is needed to implement \textquotesingle{}\doxylink{class_calendar}{Calendar} delegation\textquotesingle{} support. This class is instantiated by \doxylink{class_user}{User}.
