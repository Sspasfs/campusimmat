\doxysubsection{Description détaillée}
Calendar\+Data parser.

This class parses the \{urn\+:ietf\+:params\+:xml\+:ns\+:caldav\}calendar-\/data \doxylink{class_x_m_l}{XML} element, as defined in\+:

\href{https://tools.ietf.org/html/rfc4791\#section-9.6}{\texttt{ https\+://tools.\+ietf.\+org/html/rfc4791\#section-\/9.\+6}}

This element is used in three distinct places in the caldav spec, but in this case, this element class only implements the calendar-\/data element as it appears in a DAV\+:prop element, in a calendar-\/query or calendar-\/multiget REPORT request.
