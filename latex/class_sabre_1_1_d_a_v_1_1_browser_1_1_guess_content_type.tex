\doxysubsection{Description détaillée}
Guess\+Content\+Type plugin.

A lot of the built-\/in File objects just return application/octet-\/stream as a content-\/type by default. This is a problem for some clients, because they expect a correct contenttype.

There\textquotesingle{}s really no accurate, fast and portable way to determine the contenttype so this extension does what the rest of the world does, and guesses it based on the file extension.
