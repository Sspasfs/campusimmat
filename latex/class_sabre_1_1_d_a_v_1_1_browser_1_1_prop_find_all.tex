Handles a specific property.

This method checks whether the specified property was requested in this PROPFIND request, and if so, it will call the callback and use the return value for it\textquotesingle{}s value.

Example\+:

\$prop\+Find-\/\texorpdfstring{$>$}{>}handle(\textquotesingle{}\{DAV\+:\}displayname\textquotesingle{}, function() \{ return \textquotesingle{}hello\textquotesingle{}; \});

Note that handle will only work the first time. If null is returned, the value is ignored.
