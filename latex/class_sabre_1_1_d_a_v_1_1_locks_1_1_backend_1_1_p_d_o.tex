\doxysubsection{Description détaillée}
The \doxylink{class_lock}{Lock} manager allows you to handle all file-\/locks centrally.

This \doxylink{class_lock}{Lock} Manager stores all its data in a database. You must pass a PDO connection object in the constructor.
