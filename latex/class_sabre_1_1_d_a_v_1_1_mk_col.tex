\doxysubsection{Description détaillée}
This class represents a MKCOL operation.

MKCOL creates a new collection. MKCOL comes in two flavours\+:


\begin{DoxyEnumerate}
\item MKCOL with no body, signifies the creation of a simple collection.
\item MKCOL with a request body. This can create a collection with a specific resource type, and a set of properties that should be set on the new collection. This can be used to create caldav calendars, carddav address books, etc.
\end{DoxyEnumerate}

Property updates must always be atomic. This means that a property update must either completely succeed, or completely fail.
