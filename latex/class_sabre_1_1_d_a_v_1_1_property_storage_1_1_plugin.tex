\doxysubsection{Description détaillée}
Property\+Storage \doxylink{class_plugin}{Plugin}.

Adding this plugin to your server allows clients to store any arbitrary Web\+DAV property.

See\+: \href{http://sabre.io/dav/property-storage/}{\texttt{ http\+://sabre.\+io/dav/property-\/storage/}}

for more information.
