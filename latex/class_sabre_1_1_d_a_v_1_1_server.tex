Sets up the server.

If a Sabre\textbackslash{}\+DAV\textbackslash{}\+Tree object is passed as an argument, it will use it as the directory tree. If a Sabre\textbackslash{}\+DAV\textbackslash{}\+INode is passed, it will create a Sabre\textbackslash{}\+DAV\textbackslash{}\+Tree and use the node as the root.

If nothing is passed, a Sabre\textbackslash{}\+DAV\textbackslash{}\+Simple\+Collection is created in a Sabre\textbackslash{}\+DAV\textbackslash{}\+Tree.

If an array is passed, we automatically create a root node, and use the nodes in the array as top-\/level children.
