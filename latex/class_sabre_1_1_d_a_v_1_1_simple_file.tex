\doxysubsection{Description détaillée}
Simple\+File.

The \textquotesingle{}Simple\+File\textquotesingle{} class is used to easily add read-\/only immutable files to the directory structure. One usecase would be to add a \textquotesingle{}readme.\+txt\textquotesingle{} to a root of a webserver with some standard content.
