\doxysubsection{Description détaillée}
String utility.

This class is mainly used to implement the \textquotesingle{}text-\/match\textquotesingle{} filter, used by both the Cal\+DAV calendar-\/query REPORT, and Card\+DAV addressbook-\/query REPORT. Because they both need it, it was decided to put it in Sabre\textbackslash{}\+DAV instead.
