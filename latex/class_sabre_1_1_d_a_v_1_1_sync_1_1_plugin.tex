\doxysubsection{Description détaillée}
This plugin all Web\+DAV-\/sync capabilities to the Server.

Web\+DAV-\/sync is defined by rfc6578

The sync capabilities only work with collections that implement Sabre\textbackslash{}\+DAV\textbackslash{}\+Sync\textbackslash{}\+ISync\+Collection.
