\doxysubsection{Description détaillée}
Temporary File Filter \doxylink{class_plugin}{Plugin}.

The purpose of this filter is to intercept some of the garbage files operation systems and applications tend to generate when mounting a Web\+DAV share as a disk.

It will intercept these files and place them in a separate directory. these files are not deleted automatically, so it is advisable to delete these after they are not accessed for 24 hours.

Currently it supports\+:
\begin{DoxyItemize}
\item OS/X style resource forks and .DS\+\_\+\+Store
\item desktop.\+ini and Thumbs.\+db (windows)
\item .\texorpdfstring{$\ast$}{*}.swp (vim temporary files)
\item .dat.\texorpdfstring{$\ast$}{*} (smultron temporary files)
\end{DoxyItemize}

Additional patterns can be added, by adding on to the temporary\+File\+Patterns property.
