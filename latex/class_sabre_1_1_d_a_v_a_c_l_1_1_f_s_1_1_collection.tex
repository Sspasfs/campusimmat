\begin{DoxyItemize}
\item \textquotesingle{}privilege\textquotesingle{}, a string such as \{DAV\+:\}read or \{DAV\+:\}write. These are currently the only supported privileges
\item \textquotesingle{}principal\textquotesingle{}, a url to the principal who owns the node
\item \textquotesingle{}protected\textquotesingle{} (optional), indicating that this ACE is not allowed to be updated.
