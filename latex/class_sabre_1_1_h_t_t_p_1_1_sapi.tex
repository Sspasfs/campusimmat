\doxysubsection{Description détaillée}
PHP SAPI.

This object is responsible for\+:
\begin{DoxyEnumerate}
\item Constructing a Request object based on the current HTTP request sent to the PHP process.
\item Sending the Response object back to the client.
\end{DoxyEnumerate}

It could be said that this class provides a mapping between the Request and Response objects, and php\textquotesingle{}s\+:


\begin{DoxyItemize}
\item \$\+\_\+\+SERVER
\item \$\+\_\+\+POST
\item \$\+\_\+\+FILES
\item php\+://input
\item \doxylink{report_8contract_8php_a77b973d137fb33212e018b042df6e3e7}{echo()}
\item header()
\item php\+://output
\end{DoxyItemize}

You can choose to either call all these methods statically, but you can also instantiate this as an object to allow for polymorphism.
