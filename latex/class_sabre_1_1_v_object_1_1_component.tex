Creates a new component.

You can specify the children either in key=\texorpdfstring{$>$}{>}value syntax, in which case properties will automatically be created, or you can just pass a list of Component and Property object.

By default, a set of sensible values will be added to the component. For an i\+Calendar object, this may be something like CALSCALE\+:GREGORIAN. To ensure that this does not happen, set \$defaults to false.
