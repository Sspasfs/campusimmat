\doxysubsection{Description détaillée}
Date\+Time property.

This object represents DATE-\/\+TIME values, as defined here\+:

\href{http://tools.ietf.org/html/rfc5545\#section-3.3.4}{\texttt{ http\+://tools.\+ietf.\+org/html/rfc5545\#section-\/3.\+3.\+4}}

This particular object has a bit of hackish magic that it may also in some cases represent a DATE value. This is because it\textquotesingle{}s a common usecase to be able to change a DATE-\/\+TIME into a DATE.
