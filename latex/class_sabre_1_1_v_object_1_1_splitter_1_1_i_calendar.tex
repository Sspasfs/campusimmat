\doxysubsection{Description détaillée}
Splitter.

This class is responsible for splitting up i\+Calendar objects.

This class expects a single VCALENDAR object with one or more calendar-\/objects inside. Objects with identical UID\textquotesingle{}s will be combined into a single object.
