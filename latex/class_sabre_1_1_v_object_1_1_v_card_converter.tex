Converts a v\+Card object to a new version.

target\+Version must be one of\+: Document\+::\+VCARD21 Document\+::\+VCARD30 Document\+::\+VCARD40

Currently only 3.\+0 and 4.\+0 as input and output versions.

2.\+1 has some minor support for the input version, it\textquotesingle{}s incomplete at the moment though.

If input and output version are identical, a clone is returned.
