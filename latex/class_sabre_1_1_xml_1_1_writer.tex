\doxysubsection{Description détaillée}
The \doxylink{class_x_m_l}{XML} Writer class.

This class works exactly as PHP\textquotesingle{}s built-\/in XMLWriter, with a few additions.

Namespaces can be registered beforehand, globally. When the first element is written, namespaces will automatically be declared.

The write\+Attribute, start\+Element and write\+Element can now take a clark-\/notation element name (example\+: \{\href{http://www.w3.org/2005/Atom}{\texttt{ http\+://www.\+w3.\+org/2005/\+Atom}}\}link).

If, when writing the namespace is a known one a prefix will automatically be selected, otherwise a random prefix will be generated.

Instead of standard string values, the writer can take Element classes (as defined by this library) to delegate the serialization.

The write() method can take array structures to quickly write out simple xml trees.

\doxysubsubsection{\texorpdfstring{write()}{write()}}
{\footnotesize\ttfamily write (\begin{DoxyParamCaption}\item[{}]{\$value }\end{DoxyParamCaption})}

Writes a value to the output stream.

The following values are supported\+:
\begin{DoxyEnumerate}
\item Scalar values will be written as-\/is, as text.
\item Null values will be skipped (resulting in a short xml tag).
\item If a value is an instance of an Element class, writing will be delegated to the object.
\item If a value is an array, two formats are supported.
\end{DoxyEnumerate}

Array format 1\+: \mbox{[} "{}\{namespace\}name1"{} =\texorpdfstring{$>$}{>} "{}.."{}, "{}\{namespace\}name2"{} =\texorpdfstring{$>$}{>} "{}.."{}, \mbox{]}

One element will be created for each key in this array. The values of this array support any format this method supports (this method is called recursively).

Array format 2\+:

\mbox{[} \mbox{[} "{}name"{} =\texorpdfstring{$>$}{>} "{}\{namespace\}name1"{} "{}value"{} =\texorpdfstring{$>$}{>} "{}.."{}, "{}attributes"{} =\texorpdfstring{$>$}{>} \mbox{[} "{}attr"{} =\texorpdfstring{$>$}{>} "{}attribute value"{}, \mbox{]} \mbox{]}, \mbox{[} "{}name"{} =\texorpdfstring{$>$}{>} "{}\{namespace\}name1"{} "{}value"{} =\texorpdfstring{$>$}{>} "{}.."{}, "{}attributes"{} =\texorpdfstring{$>$}{>} \mbox{[} "{}attr"{} =\texorpdfstring{$>$}{>} "{}attribute value"{}, \mbox{]} \mbox{]} \mbox{]}
