Embed the enclosure using Javascript

{\ttfamily \$options} is an array or comma-\/separated key\+:value string, with the following properties\+:


\begin{DoxyItemize}
\item {\ttfamily alt} (string)\+: Alternate content for when an end-\/user does not have the appropriate handler installed or when a file type is unsupported. Can be any text or HTML. Defaults to blank.
\item {\ttfamily altclass} (string)\+: If a file type is unsupported, the end-\/user will see the alt text (above) linked directly to the content. That link will have this value as its class name. Defaults to blank.
\item {\ttfamily audio} (string)\+: This is an image that should be used as a placeholder for audio files before they\textquotesingle{}re loaded (Quick\+Time-\/only). Can be any relative or absolute URL. Defaults to blank.
\item {\ttfamily bgcolor} (string)\+: The background color for the media, if not already transparent. Defaults to {\ttfamily \#ffffff}.
\item {\ttfamily height} (integer)\+: The height of the embedded media. Accepts any numeric pixel value (such as {\ttfamily 360}) or {\ttfamily auto}. Defaults to {\ttfamily auto}, and it is recommended that you use this default.
\item {\ttfamily loop} (boolean)\+: Do you want the media to loop when it\textquotesingle{}s done? Defaults to {\ttfamily false}.
\item {\ttfamily mediaplayer} (string)\+: The location of the included {\ttfamily mediaplayer.\+swf} file. This allows for the playback of Flash Video ({\ttfamily .flv}) files, and is the default handler for non-\/\+Odeo MP3\textquotesingle{}s. Defaults to blank.
\item {\ttfamily video} (string)\+: This is an image that should be used as a placeholder for video files before they\textquotesingle{}re loaded (Quick\+Time-\/only). Can be any relative or absolute URL. Defaults to blank.
\item {\ttfamily width} (integer)\+: The width of the embedded media. Accepts any numeric pixel value (such as {\ttfamily 480}) or {\ttfamily auto}. Defaults to {\ttfamily auto}, and it is recommended that you use this default.
\item {\ttfamily widescreen} (boolean)\+: Is the enclosure widescreen or standard? This applies only to video enclosures, and will automatically resize the content appropriately. Defaults to {\ttfamily false}, implying 4\+:3 mode.
\end{DoxyItemize}

Note\+: Non-\/widescreen (4\+:3) mode with {\ttfamily width} and {\ttfamily height} set to {\ttfamily auto} will default to 480x360 video resolution. Widescreen (16\+:9) mode with {\ttfamily width} and {\ttfamily height} set to {\ttfamily auto} will default to 480x270 video resolution.
