\doxysubsection{Description détaillée}
The \doxylink{class_spl_fixed_array}{Spl\+Fixed\+Array} class provides the main functionalities of array. The main differences between a \doxylink{class_spl_fixed_array}{Spl\+Fixed\+Array} and a normal PHP array is that the \doxylink{class_spl_fixed_array}{Spl\+Fixed\+Array} is of fixed length and allows only integers within the range as indexes. The advantage is that it allows a faster array implementation.
