Using a Memcached\+Adapter with a Tag\+Aware\+Adapter for storing tags is discouraged. Using a Redis\+Adapter is recommended instead. If you cannot do otherwise, be aware that\+:
\begin{DoxyItemize}
\item the Memcached\+::\+OPT\+\_\+\+BINARY\+\_\+\+PROTOCOL must be enabled (that\textquotesingle{}s the default when using Memcached\+Adapter\+::create\+Connection());
\item tags eviction by Memcached\textquotesingle{}s LRU algorithm will break by-\/tags invalidation; your Memcached memory should be large enough to never trigger LRU.
\end{DoxyItemize}

Using a Memcached\+Adapter as a pure items store is fine.
