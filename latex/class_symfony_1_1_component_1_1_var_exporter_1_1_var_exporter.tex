\doxysubsection{Description détaillée}
Exports serializable PHP values to PHP code.

Var\+Exporter allows serializing PHP data structures to plain PHP code (like var\+\_\+export()) while preserving all the semantics associated with serialize() (unlike var\+\_\+export()).

By leveraging OPcache, the generated PHP code is faster than doing the same with unserialize().
