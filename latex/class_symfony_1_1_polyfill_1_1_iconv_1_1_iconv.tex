\doxysubsection{Description détaillée}
iconv implementation in pure PHP, UTF-\/8 centric.

Implemented\+:
\begin{DoxyItemize}
\item iconv -\/ Convert string to requested character encoding
\item iconv\+\_\+mime\+\_\+decode -\/ Decodes a MIME header field
\item iconv\+\_\+mime\+\_\+decode\+\_\+headers -\/ Decodes multiple MIME header fields at once
\item iconv\+\_\+get\+\_\+encoding -\/ Retrieve internal configuration variables of iconv extension
\item iconv\+\_\+set\+\_\+encoding -\/ Set current setting for character encoding conversion
\item iconv\+\_\+mime\+\_\+encode -\/ Composes a MIME header field
\item iconv\+\_\+strlen -\/ Returns the character count of string
\item iconv\+\_\+strpos -\/ Finds position of first occurrence of a needle within a haystack
\item iconv\+\_\+strrpos -\/ Finds the last occurrence of a needle within a haystack
\item iconv\+\_\+substr -\/ Cut out part of a string
\end{DoxyItemize}

Charsets available for conversion are defined by files in the charset/ directory and by Iconv\+::\$alias below. You\textquotesingle{}re welcome to send back any addition you make.
