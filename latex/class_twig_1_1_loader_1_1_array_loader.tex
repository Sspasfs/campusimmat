\doxysubsection{Description détaillée}
Loads a template from an array.

When using this loader with a cache mechanism, you should know that a new cache key is generated each time a template content "{}changes"{} (the cache key being the source code of the template). If you don\textquotesingle{}t want to see your cache grows out of control, you need to take care of clearing the old cache file by yourself.

This loader should only be used for unit testing.
