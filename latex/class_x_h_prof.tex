Il you need to "{}profile"{} some part of code

Install the pecl/xhprof extension

Add XHPROF\+\_\+\+PATH and XHPROF\+\_\+\+URL in config/local\+\_\+define.\+php (if needed)

Before the code \$prof = new \doxylink{class_x_h_prof}{XHProf}("{}something useful"{});

If the code contains an \doxylink{report_8contract_8php_a77b973d137fb33212e018b042df6e3e7}{exit()} or a redirect() you must also call (before) unset(\$prof);

php-\/errors.\+log will give you the URL of the result.
