Return configuration for a service manager instance as an array.

Implementations MUST return an array compatible with Service\+Manager\+::configure, containing one or more of the following keys\+:


\begin{DoxyItemize}
\item abstract\+\_\+factories
\item aliases
\item delegators
\item factories
\item initializers
\item invokables
\item lazy\+\_\+services
\item services
\item shared
\end{DoxyItemize}

In other words, this should return configuration that can be used to instantiate a service manager or plugin manager, or pass to its {\ttfamily \doxylink{report_8contract_8php_a77b973d137fb33212e018b042df6e3e7}{with\+Config()}} method.
