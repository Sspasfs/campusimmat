\doxysubsection{Description détaillée}
Hint to the laminas-\/modulemanager Service\+Listener that a module provides validators.

Module classes implementing this interface hint to Laminas\textbackslash{}\+Module\+Manager\textbackslash{}\+Service\+Listener that they provide validators for the Validator\+Plugin\+Manager.

For users of laminas-\/mvc/laminas-\/modulemanager v2, this poses no backwards-\/compatibility break as the method get\+Validator\+Config is still duck-\/typed within Laminas\textbackslash{}\+Validator\textbackslash{}\+Module when providing configuration to the Service\+Listener.
