\doxysubsection{Description détaillée}
This node represents a single notification.

The signature is mostly identical to that of Sabre\textbackslash{}\+DAV\textbackslash{}\+IFile, but the get() method MUST return an xml document that matches the requirements of the \textquotesingle{}caldav-\/notifications.\+txt\textquotesingle{} spec.

For a complete example, check out the \doxylink{class_notification}{Notification} class, which contains some helper functions.
