\doxysubsection{Description détaillée}
IMulti\+Get.

This interface adds a tiny bit of functionality to collections.

There a certain situations, in particular in relation to Web\+DAV-\/\+Sync, Cal\+DAV and Card\+DAV, where information for a list of items will be requested.

Because the get\+Child() call is the main abstraction method, this can in reality result in many database calls, which could potentially be optimized.

The Multi\+Get interface is used by the server in these cases.
