\doxysubsection{Description détaillée}
This is the \doxylink{class_x_m_l}{XML} element interface.

Elements are responsible for serializing and deserializing part of an \doxylink{class_x_m_l}{XML} document into PHP values.

It combines Xml\+Serializable and Xml\+Deserializable into one logical class that does both.
