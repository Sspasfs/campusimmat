Fetches a value from the pool or computes it if not found.

On cache misses, a callback is called that should return the missing value. This callback is given a PSR-\/6 Cache\+Item\+Interface instance corresponding to the requested key, that could be used e.\+g. for expiration control. It could also be an Item\+Interface instance when its additional features are needed.
