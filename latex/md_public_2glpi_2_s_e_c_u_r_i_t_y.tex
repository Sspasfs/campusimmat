 {\bfseries{⚠️ Please never use standard issues to report security problems; vulnerabilities are published once a fix release is available. ⚠️}}\hypertarget{md_public_2glpi_2_s_e_c_u_r_i_t_y_autotoc_md3054}{}\doxysection{\texorpdfstring{Reporting a Vulnerability}{Reporting a Vulnerability}}\label{md_public_2glpi_2_s_e_c_u_r_i_t_y_autotoc_md3054}
If you found a security issue, please contact us by\+:


\begin{DoxyItemize}
\item a mail to \mbox{[}glpi-\/security AT ow2.\+org\mbox{]}
\item open an advisory on \href{https://github.com/glpi-project/glpi/security/advisories/new}{\texttt{ github}}
\end{DoxyItemize}

You should provide us all details about the issue and the way to reproduce it. You may also provide a script that can be used to check the issue exists.

Once the report will be handled, and if the issue is not yet fixed (or in progress) we\textquotesingle{}ll add it to the Git\+Hub security tab, and add you as observer. Meanwhile, you will reserve a CVE for the issue.
