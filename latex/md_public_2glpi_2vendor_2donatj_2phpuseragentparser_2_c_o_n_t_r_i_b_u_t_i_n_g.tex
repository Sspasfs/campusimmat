Issues can be reported via the \href{https://github.com/donatj/PhpUserAgent/issues}{\texttt{ Github Issues}} page.


\begin{DoxyItemize}
\item {\bfseries{Detail is key}}\+: If a browser is being misidentified, one or more sample user agent strings are key to getting it resolved.
\item {\bfseries{Missing Browser}}\+: Is it modern? What is it being misidentified as? There are a lot of dead browsers out there that there is no reason to support.
\end{DoxyItemize}

Please do not file any requests for OS version identification. It is not a desired feature.\hypertarget{md_public_2glpi_2vendor_2donatj_2phpuseragentparser_2_c_o_n_t_r_i_b_u_t_i_n_g_autotoc_md5751}{}\doxysection{\texorpdfstring{Pull Requests}{Pull Requests}}\label{md_public_2glpi_2vendor_2donatj_2phpuseragentparser_2_c_o_n_t_r_i_b_u_t_i_n_g_autotoc_md5751}
Pull requests are truly appreciated. While I try my best to stay on top of browsers hitting the market it is still a difficult task.


\begin{DoxyItemize}
\item {\bfseries{Formatting}}\+: Indentation {\bfseries{must}} use tabs. Please try to match internal formatting and spacing to existing code.
\item {\bfseries{\doxylink{namespace_tests}{Tests}}}\+: If you\textquotesingle{}re adding support for a new browser be sure to add test user agents for if at all possible {\itshape {\bfseries{every platform}}} the browser is available on. Untested code will take much longer to be merged.
\item {\bfseries{Terseness}}\+: Try to be terse. Be clever. Take up as little space as possible. The point of this project initially was to be smaller than the other guys. 
\end{DoxyItemize}
