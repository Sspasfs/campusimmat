We accept contributions via Pull Requests on \href{https://github.com/thephpleague/oauth2-google}{\texttt{ Github}}.\hypertarget{md_public_2glpi_2vendor_2league_2oauth2-google_2_c_o_n_t_r_i_b_u_t_i_n_g_autotoc_md6742}{}\doxysection{\texorpdfstring{Pull Requests}{Pull Requests}}\label{md_public_2glpi_2vendor_2league_2oauth2-google_2_c_o_n_t_r_i_b_u_t_i_n_g_autotoc_md6742}

\begin{DoxyItemize}
\item {\bfseries{\href{https://github.com/php-fig/fig-standards/blob/master/accepted/PSR-2-coding-style-guide.md}{\texttt{ PSR-\/2 Coding Standard}}}} -\/ The easiest way to apply the conventions is to install \href{http://pear.php.net/package/PHP_CodeSniffer}{\texttt{ PHP Code Sniffer}}.
\item {\bfseries{Add tests!}} -\/ Your patch won\textquotesingle{}t be accepted if it doesn\textquotesingle{}t have tests.
\item {\bfseries{\doxylink{class_document}{Document} any change in behaviour}} -\/ Make sure the README and any other relevant documentation are kept up-\/to-\/date.
\item {\bfseries{Consider our release cycle}} -\/ We try to follow Sem\+Ver. Randomly breaking public APIs is not an option.
\item {\bfseries{Create topic branches}} -\/ Don\textquotesingle{}t ask us to pull from your master branch.
\item {\bfseries{One pull request per feature}} -\/ If you want to do more than one thing, send multiple pull requests.
\item {\bfseries{Send coherent history}} -\/ Make sure each individual commit in your pull request is meaningful. If you had to make multiple intermediate commits while developing, please squash them before submitting.
\item {\bfseries{Ensure tests pass!}} -\/ Please run the tests (see below) before submitting your pull request, and make sure they pass. We won\textquotesingle{}t accept a patch until all tests pass.
\item {\bfseries{Ensure no coding standards violations}} -\/ Please run PHP Code Sniffer using the PSR-\/2 standard (see below) before submitting your pull request. A violation will cause the build to fail, so please make sure there are no violations. We can\textquotesingle{}t accept a patch if the build fails.
