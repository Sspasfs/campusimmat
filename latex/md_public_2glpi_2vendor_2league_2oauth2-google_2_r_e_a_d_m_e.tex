This package provides Google OAuth 2.\+0 support for the PHP League\textquotesingle{}s \href{https://github.com/thephpleague/oauth2-client}{\texttt{ OAuth 2.\+0 Client}}.

This package is compliant with \href{https://github.com/php-fig/fig-standards/blob/master/accepted/PSR-1-basic-coding-standard.md}{\texttt{ PSR-\/1}}, \href{https://github.com/php-fig/fig-standards/blob/master/accepted/PSR-2-coding-style-guide.md}{\texttt{ PSR-\/2}} and \href{https://github.com/php-fig/fig-standards/blob/master/accepted/PSR-4-autoloader.md}{\texttt{ PSR-\/4}}. If you notice compliance oversights, please send a patch via pull request.\hypertarget{md_public_2glpi_2vendor_2league_2oauth2-google_2_r_e_a_d_m_e_autotoc_md6746}{}\doxysection{\texorpdfstring{Requirements}{Requirements}}\label{md_public_2glpi_2vendor_2league_2oauth2-google_2_r_e_a_d_m_e_autotoc_md6746}
The following versions of PHP are supported.


\begin{DoxyItemize}
\item PHP 7.\+2
\item PHP 7.\+3
\item PHP 7.\+4
\item PHP 8.\+0
\end{DoxyItemize}

This package uses \href{https://developers.google.com/identity/protocols/OpenIDConnect}{\texttt{ Open\+ID Connect}} to authenticate users with Google accounts.

To use this package, it will be necessary to have a Google client ID and client secret. These are referred to as {\ttfamily \{google-\/client-\/id\}} and {\ttfamily \{google-\/client-\/secret\}} in the documentation.

Please follow the \href{https://developers.google.com/identity/protocols/OpenIDConnect\#registeringyourapp}{\texttt{ Google instructions}} to create the required credentials.\hypertarget{md_public_2glpi_2vendor_2league_2oauth2-google_2_r_e_a_d_m_e_autotoc_md6747}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md_public_2glpi_2vendor_2league_2oauth2-google_2_r_e_a_d_m_e_autotoc_md6747}

\begin{DoxyCode}{0}
\DoxyCodeLine{composer\ require\ league/oauth2-\/google}

\hypertarget{md_public_2glpi_2vendor_2league_2oauth2-google_2_r_e_a_d_m_e_autotoc_md6750}{}\doxysubsubsection{\texorpdfstring{Available Options}{Available Options}}\label{md_public_2glpi_2vendor_2league_2oauth2-google_2_r_e_a_d_m_e_autotoc_md6750}
The {\ttfamily Google} provider has the following \href{https://developers.google.com/identity/protocols/OpenIDConnect\#authenticationuriparameters}{\texttt{ options}}\+:


\begin{DoxyItemize}
\item {\ttfamily access\+Type} to use online or offline access
\item {\ttfamily hosted\+Domain} to authenticate G Suite users
\item {\ttfamily prompt} to modify the prompt that the user will see
\item {\ttfamily scopes} to request access to additional user information
\end{DoxyItemize}\hypertarget{md_public_2glpi_2vendor_2league_2oauth2-google_2_r_e_a_d_m_e_autotoc_md6751}{}\doxysubsubsection{\texorpdfstring{Accessing Token JWT}{Accessing Token JWT}}\label{md_public_2glpi_2vendor_2league_2oauth2-google_2_r_e_a_d_m_e_autotoc_md6751}
Google provides a \href{https://jwt.io/}{\texttt{ JSON Web Token}} (JWT) with all access tokens. This token \href{https://developers.google.com/identity/protocols/OpenIDConnect\#obtainuserinfo}{\texttt{ contains basic information}} about the authenticated user. The JWT can be accessed from the {\ttfamily id\+\_\+token} value of the access token\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{}
\DoxyCodeLine{\mbox{\hyperlink{report_8contract_8php_affc45c6ace2eeb3f300b054dbf9592b6}{\$values}}\ =\ \$token-\/>getValues();}
\DoxyCodeLine{}
\DoxyCodeLine{\mbox{\hyperlink{report_8contract_8php_a77b973d137fb33212e018b042df6e3e7}{\$jwt}}\ =\ \mbox{\hyperlink{report_8contract_8php_affc45c6ace2eeb3f300b054dbf9592b6}{\$values}}[\textcolor{stringliteral}{'id\_token'}];}

\end{DoxyCode}


Parsing the JWT will require a \href{https://packagist.org/search/?q=jwt}{\texttt{ JWT parser}}.
