\hypertarget{md_public_2glpi_2vendor_2mexitek_2phpcolors_2_r_e_a_d_m_e_autotoc_md6762}{}\doxysection{\texorpdfstring{How it works}{How it works}}\label{md_public_2glpi_2vendor_2mexitek_2phpcolors_2_r_e_a_d_m_e_autotoc_md6762}
Instantiate an object of the color class with a hex color string {\ttfamily \$foo = new Color("{}336699"{})}. That\textquotesingle{}s it! Now, call the methods you need for different color variants.\hypertarget{md_public_2glpi_2vendor_2mexitek_2phpcolors_2_r_e_a_d_m_e_autotoc_md6763}{}\doxysection{\texorpdfstring{Available Methods}{Available Methods}}\label{md_public_2glpi_2vendor_2mexitek_2phpcolors_2_r_e_a_d_m_e_autotoc_md6763}

\begin{DoxyItemize}
\item {\bfseries{darken( \mbox{[}\$amount\mbox{]} )}} \+: Allows you to obtain a darker shade of your color. Optionally you can decide to darken using a desired percentage.
\item {\bfseries{lighten( \mbox{[}\$amount\mbox{]} )}} \+: Allows you to obtain a lighter shade of your color. Optionally you can decide to lighten using a desired percentage.
\item {\bfseries{mix(\$hex, \mbox{[}\$amount\mbox{]} )}} \+: Allows you to mix another color to your color. Optionally you can decide to set the percent of second color or original color amount is ranged -\/100...0...100.
\item {\bfseries{is\+Light( \mbox{[}\$hex\mbox{]} )}} \+: Determins whether your color (or the provide param) is considered a "{}light"{} color. Returns {\ttfamily TRUE} if color is light.
\item {\bfseries{is\+Dark( \mbox{[}\$hex\mbox{]} )}} \+: Determins whether your color (or the provide param) is considered a "{}dark"{} color. Returns {\ttfamily TRUE} if color is dark.
\item {\bfseries{make\+Gradient( \mbox{[}\$amount\mbox{]} )}} \+: Returns an array with 2 indices {\ttfamily light} and {\ttfamily dark}, the initial color will either be selected for {\ttfamily light} or {\ttfamily dark} depending on its brightness, then the other color will be generated. The optional param allows for a static lighten or darkened amount.
\item {\bfseries{complementary()}} \+: Returns the color "{}opposite"{} or complementary to your color.
\item {\bfseries{get\+Hex()}} \+: Returns the original hex color.
\item {\bfseries{get\+Hsl()}} \+: Returns HSL array for your color.
\item {\bfseries{get\+Rgb()}} \+: Returns RGB array for your color.
\end{DoxyItemize}
