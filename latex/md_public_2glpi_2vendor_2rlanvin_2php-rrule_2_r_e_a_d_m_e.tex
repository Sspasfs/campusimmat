I started this library because I wasn\textquotesingle{}t happy with the existing implementations in PHP, so I thought it would be a good learning project to port the python-\/dateutil rrule implementation into PHP.

The Python lib was a bit difficult to understand because the algorithms are not commented and the variables are very opaque (I\textquotesingle{}m looking at you {\ttfamily lno1wkst}). I tried to comment and explain as much of the algorithm as possible in this PHP port, so feel free to check the code if you\textquotesingle{}re interested.

The lib differs from the python version in various aspects, notably in the respect of the RFC. This version is a bit strictier and will not accept many non-\/compliant combinations of rule parts, that the python version otherwise accepts. There are also some additional features in this version.\hypertarget{md_public_2glpi_2vendor_2rlanvin_2php-rrule_2_r_e_a_d_m_e_autotoc_md7043}{}\doxysection{\texorpdfstring{License}{License}}\label{md_public_2glpi_2vendor_2rlanvin_2php-rrule_2_r_e_a_d_m_e_autotoc_md7043}
