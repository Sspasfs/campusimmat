The \doxylink{class_v_object}{VObject} library allows you to easily parse and manipulate \href{https://tools.ietf.org/html/rfc5545}{\texttt{ i\+Calendar}} and \href{https://tools.ietf.org/html/rfc6350}{\texttt{ v\+Card}} objects using PHP.

The goal of the \doxylink{class_v_object}{VObject} library is to create a very complete library, with an easy-\/to-\/use API.\hypertarget{md_public_2glpi_2vendor_2sabre_2vobject_2_r_e_a_d_m_e_autotoc_md7102}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md_public_2glpi_2vendor_2sabre_2vobject_2_r_e_a_d_m_e_autotoc_md7102}
Make sure you have \href{https://getcomposer.org/}{\texttt{ Composer}} installed, and then run\+: \begin{DoxyVerb}composer require sabre/vobject "^4.0"
\end{DoxyVerb}
 This package requires PHP 5.\+5. If you need the PHP 5.\+3/5.4 version of this package instead, use\+:

\begin{DoxyVerb}composer require sabre/vobject "^3.4"
\end{DoxyVerb}
