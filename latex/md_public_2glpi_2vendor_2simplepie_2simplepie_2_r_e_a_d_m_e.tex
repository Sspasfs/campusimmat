\chapter{Simple\+Pie}
\hypertarget{md_public_2glpi_2vendor_2simplepie_2simplepie_2_r_e_a_d_m_e}{}\label{md_public_2glpi_2vendor_2simplepie_2simplepie_2_r_e_a_d_m_e}\index{SimplePie@{SimplePie}}
\doxylink{class_simple_pie}{Simple\+Pie} is a very fast and easy-\/to-\/use class, written in PHP, that puts the \textquotesingle{}simple\textquotesingle{} back into \textquotesingle{}really simple syndication\textquotesingle{}. Flexible enough to suit beginners and veterans alike, \doxylink{class_simple_pie}{Simple\+Pie} is focused on \href{http://simplepie.org/wiki/faq/what_is_simplepie}{\texttt{ speed, ease of use, compatibility and standards compliance}}.\hypertarget{md_public_2glpi_2vendor_2simplepie_2simplepie_2_r_e_a_d_m_e_autotoc_md7189}{}\doxysection{\texorpdfstring{Requirements}{Requirements}}\label{md_public_2glpi_2vendor_2simplepie_2simplepie_2_r_e_a_d_m_e_autotoc_md7189}

\begin{DoxyItemize}
\item PHP 5.\+6+ (Required since \doxylink{class_simple_pie}{Simple\+Pie} 1.\+5.\+3)
\item libxml2 (certain 2.\+7.\+x releases are too buggy for words, and will crash)
\item One of iconv, mbstring or intl extensions
\item c\+URL or fsockopen()
\item PCRE support
\end{DoxyItemize}\hypertarget{md_public_2glpi_2vendor_2simplepie_2simplepie_2_r_e_a_d_m_e_autotoc_md7190}{}\doxysection{\texorpdfstring{What comes in the package?}{What comes in the package?}}\label{md_public_2glpi_2vendor_2simplepie_2simplepie_2_r_e_a_d_m_e_autotoc_md7190}

Since \doxylink{class_simple_pie}{Simple\+Pie} 1.\+3, we\textquotesingle{}ve split the classes into separate files to make it easier to maintain and use.

If you\textquotesingle{}d like a single monolithic file, you can run {\ttfamily php build/compile.\+php} to generate {\ttfamily Simple\+Pie.\+compiled.\+php}.\hypertarget{md_public_2glpi_2vendor_2simplepie_2simplepie_2_r_e_a_d_m_e_autotoc_md7192}{}\doxysection{\texorpdfstring{To start the demo}{To start the demo}}\label{md_public_2glpi_2vendor_2simplepie_2simplepie_2_r_e_a_d_m_e_autotoc_md7192}
