This file describes all Unified\+Archive API.

Unified\+Archive is represented by few basic classes under {\ttfamily \textbackslash{}wapmorgan\textbackslash{}Unified\+Archive} namespace\+:


\begin{DoxyEnumerate}
\item \`{}\+Formats\`{} keeps information about formats support and specific format functions.
\item \`{}\+Unified\+Archive\`{} -\/ represents an archive and provides related functions.
\item \`{}\+Archive\+Entry\`{} -\/ represents information about a specific file from archive. This object can be obtained by call to one of {\ttfamily Unified\+Archive} methods.
\end{DoxyEnumerate}\hypertarget{md_public_2glpi_2vendor_2wapmorgan_2unified-archive_2docs_2_a_p_i_autotoc_md7487}{}\doxysection{\texorpdfstring{Formats}{Formats}}\label{md_public_2glpi_2vendor_2wapmorgan_2unified-archive_2docs_2_a_p_i_autotoc_md7487}
{\ttfamily \$format} should be one of {\ttfamily Formats} constants (such as {\ttfamily Formats\+::\+ZIP} and so on). Full list of constants provided in the \doxylink{report_8contract_8php_a77b973d137fb33212e018b042df6e3e7}{appendix of this document}. {\itshape If you want to enabled specific format support, you need to install an additional program or php extension. List of extensions that should be installed can be obtained by executing built-\/in {\ttfamily cam}\+: {\ttfamily ./vendor/bin/cam system\+:drivers}}

All methods are static.

Creates an archive with given {\ttfamily \$files} list. {\ttfamily \$files} is an array of files or directories. If file/directory passed with numeric key (e.\+g `\mbox{[}\textquotesingle{}file', \textquotesingle{}directory\textquotesingle{}\mbox{]}{\ttfamily ), then file/directory will have it\textquotesingle{}s full path in an archive. If file/directory is a key (e.\+g}\mbox{[}\textquotesingle{}in\+\_\+archive\+\_\+path\textquotesingle{} =\texorpdfstring{$>$}{>} \textquotesingle{}file1\textquotesingle{}\mbox{]}\`{}), then file/directory will have path as it\textquotesingle{}s value. In case of success, number of stored files will be returned.
