\doxysubsection{Description détaillée}
Class to create QR-\/code arrays for \doxylink{class_t_c_p_d_f}{TCPDF} class. QR Code symbol is a 2D barcode that can be scanned by handy terminals such as a mobile phone with CCD. The capacity of QR Code is up to 7000 digits or 4000 characters, and has high robustness. This class supports QR Code model 2, described in JIS (Japanese Industrial Standards) X0510\+:2004 or ISO/\+IEC 18004. Currently the following features are not supported\+: ECI and FNC1 mode, Micro QR Code, QR Code model 1, Structured mode.
