Relation constants between tables.

This mapping is used for to detect links between objects. For example, it is used to detect if items are associated to another item that is going to be deleted, in order to prevent deletion or ask for user what to do with linked items.

Format is\+: \mbox{[} \textquotesingle{}referenced\+\_\+table\+\_\+name\textquotesingle{} =\texorpdfstring{$>$}{>} \mbox{[} \textquotesingle{}linked\+\_\+table\+\_\+name\+\_\+1\textquotesingle{} =\texorpdfstring{$>$}{>} \textquotesingle{}foreign\+\_\+key\+\_\+1\textquotesingle{}, \textquotesingle{}linked\+\_\+table\+\_\+name\+\_\+2\textquotesingle{} =\texorpdfstring{$>$}{>} \mbox{[}\textquotesingle{}foreign\+\_\+key\+\_\+2\textquotesingle{}, \textquotesingle{}foreign\+\_\+key\+\_\+3\textquotesingle{}\mbox{]}, \textquotesingle{}linked\+\_\+table\+\_\+name\+\_\+2\textquotesingle{} =\texorpdfstring{$>$}{>} \mbox{[}\mbox{[}\textquotesingle{}items\+\_\+id\textquotesingle{}, \textquotesingle{}itemtype\textquotesingle{}\mbox{]}\mbox{]}, \mbox{]} \mbox{]} where\+:
\begin{DoxyItemize}
\item \textquotesingle{}referenced\+\_\+table\+\_\+name\textquotesingle{} is the name of a table having its id referenced in other tables,
\item \textquotesingle{}linked\+\_\+table\+\_\+name\+\_\+\texorpdfstring{$\ast$}{*}\textquotesingle{} is the name of a table that have foreign keys referencing the table \textquotesingle{}referenced\+\_\+table\+\_\+name\textquotesingle{},
\item \textquotesingle{}foreign\+\_\+key\+\_\+\texorpdfstring{$\ast$}{*}\textquotesingle{} is the name of the field that is a foreign key (can be \mbox{[}\textquotesingle{}items\+\_\+id\textquotesingle{}, \textquotesingle{}itemtype\textquotesingle{}\mbox{]}).
\end{DoxyItemize}

/!\textbackslash{} "{}\+\_\+"{} prefix is used to disable usage check on relations while deleting an item when they are handled by application. Applications handle specific usage check and links updates \+:
\begin{DoxyItemize}
\item in {\ttfamily Common\+DBTM\+::clean\+Relation\+Table()} method,
\item in {\ttfamily \$item\+::clean\+DBon\+Purge()} method,
\item using {\ttfamily \$forward\+\_\+entity\+\_\+to} values,
\item by {\ttfamily \doxylink{class_common_tree_dropdown}{Common\+Tree\+Dropdown}} logic for recursive keys. Relations will still be used to check ability to disable recursivity on an element.
\end{DoxyItemize}
