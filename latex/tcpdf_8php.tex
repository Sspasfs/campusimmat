\doxysubsection{Description détaillée}
This is a PHP class for generating PDF documents without requiring external extensions.~\newline
 \doxylink{class_t_c_p_d_f}{TCPDF} project (\href{http://www.tcpdf.org}{\texttt{ http\+://www.\+tcpdf.\+org}}) was originally derived in 2002 from the Public \doxylink{class_domain}{Domain} FPDF class by Olivier Plathey (\href{http://www.fpdf.org}{\texttt{ http\+://www.\+fpdf.\+org}}), but now is almost entirely rewritten.~\newline
 \doxysubsubsubsection*{\doxylink{class_t_c_p_d_f}{TCPDF} main features are\+:}


\begin{DoxyItemize}
\item no external libraries are required for the basic functions; 
\item all standard page formats, custom page formats, custom margins and units of measure; 
\item UTF-\/8 Unicode and Right-\/\+To-\/\+Left languages; 
\item True\+Type\+Unicode, True\+Type, Type1 and CID-\/0 fonts; 
\item font subsetting; 
\item methods to publish some XHTML + CSS code, Javascript and Forms; 
\item images, graphic (geometric figures) and transformation methods; 
\item supports JPEG, PNG and SVG images natively, all images supported by GD (GD, GD2, GD2\+PART, GIF, JPEG, PNG, BMP, XBM, XPM) and all images supported via Image\+Magick (\href{http://www.imagemagick.org/www/formats.html}{\texttt{ http\+://www.\+imagemagick.\+org/www/formats.\+html}}) 
\item 1D and 2D barcodes\+: CODE 39, ANSI MH10.\+8M-\/1983, USD-\/3, 3 of 9, CODE 93, USS-\/93, Standard 2 of 5, Interleaved 2 of 5, CODE 128 A/\+B/C, 2 and 5 Digits UPC-\/\+Based Extension, EAN 8, EAN 13, UPC-\/A, UPC-\/E, MSI, POSTNET, PLANET, RMS4\+CC (Royal Mail 4-\/state Customer Code), CBC (Customer Bar Code), KIX (Klant index -\/ Customer index), Intelligent Mail Barcode, Onecode, USPS-\/\+B-\/3200, CODABAR, CODE 11, PHARMACODE, PHARMACODE TWO-\/\+TRACKS, \doxylink{class_datamatrix}{Datamatrix}, QR-\/\+Code, \doxylink{class_p_d_f417}{PDF417}; 
\item JPEG and PNG ICC profiles, Grayscale, RGB, CMYK, Spot Colors and Transparencies; 
\item automatic page header and footer management; 
\item document encryption up to 256 bit and digital signature certifications; 
\item transactions to UNDO commands; 
\item PDF annotations, including links, text and file attachments; 
\item text rendering modes (fill, stroke and clipping); 
\item multiple columns mode; 
\item no-\/write page regions; 
\item bookmarks, named destinations and table of content; 
\item text hyphenation; 
\item text stretching and spacing (tracking); 
\item automatic page break, line break and text alignments including justification; 
\item automatic page numbering and page groups; 
\item move and delete pages; 
\item page compression (requires php-\/zlib extension); 
\item XOBject Templates; 
\item Layers and object visibility; 
\item PDF/\+A-\/1b support. 
\end{DoxyItemize}Tools to encode your unicode fonts are on fonts/utils directory.
